Glossary



Area

One web page is divided in Areas such as Header, Navigation, Main Content, Sections and Footer.
IMPORTANT: The amount of areas inside one page is limited, usually something around 8 or 10 areas per page.

Widget

Widget is an element that you can add into one Area. Each widget created can have multiple contents (elements). For example, an Activities or Sport's Activities Widget can store tons of activities.
IMPORTANT: The amount of widgets are almost unlimited, where you could have hundreds and pick up 10 of them to fit over the areas available.

Widget Type

Widgets has a type such as SimpleContent, LongContent, IconContent, Background or PageLink. Each type has small differences, for example, SimpleContent uses simple text and squared images when LongContent uses WYSIWYG and rectangular images. IconContent used icons instead of images

Some widgets such as Background and PageLink have special fields like URL and don't use simple text neigher WYSIWYG.

Widget Style

Style is about visual, how those widget's elements are displayed. It's possible to show just one element or multiple elements following 2, 3 or 4 elements per line.

Page Widget

It's a special Widget that besides be pluggable inside an area, it has its own separated page. For example, the Newsfeeds Widget that might contains heaps of elements and be displayed over a separated page is more convinient.
